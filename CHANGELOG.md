# Changelog

## 1.2.0 - 2026-06-10

- Added direct-link Razuna Gallery block with multi-select image picking.
- Added managed WordPress imports for workflows that need attachment IDs.
- Added Razuna import actions for core media blocks, featured images, the Media Library and media modal flows.
- Stored source Razuna metadata on imported WordPress attachments.
- Kept the WordPress.org review fixes for plugin URL, settings-page JavaScript enqueueing and WPCS cleanup.
- Updated plugin/readme stable version to 1.2.0.

## 1.1.1 - 2026-06-09

- Fixed image dimensions for thumbnail and saved-format inserts.
- Added dimension parsing from format metadata, labels and direct-link URL parameters.
- Preserved width and height in classic-editor direct-link inserts.

## 1.1.0 - 2026-06-05

- Kept "My workspace" first and sorted remaining workspaces alphabetically.
- Added paged browsing and search results to the asset picker.
- Added infinite scroll with a Load more fallback for larger libraries.
- Improved asset option labels with dimensions, file sizes and saved format details.
- Updated plugin URI and stable version to 1.1.0.

## 1.0.0 - 2026-06-04

- Initial Razuna DAM WordPress plugin release.
- Added OAuth connection flow with server-side token handling.
- Added asset browser/search, workspace/folder navigation and saved-format selection.
- Added direct-link embeds, download links, Razuna Asset block and classic-editor inserter.
- Added WordPress.org release packaging and GitHub release ZIP workflow.
